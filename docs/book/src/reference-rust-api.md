## How to navigate rustdoc

- Type any symbol name into the search box at the top
- The sidebar groups by `Modules`, `Structs`, `Enums`, `Traits`, `Functions`
- Source code links: click the `[src]` tag on any item to jump to the underlying `.rs`

## Building locally

```bash
cargo doc --no-deps --workspace --release --open
```

`--no-deps` excludes third-party crates (much faster build).
`--workspace` includes every in-tree crate.

## Notes

- WASM-only crates (`hymeko_wasm`) are excluded from the published rustdoc since they require the `wasm32-unknown-unknown` target. To inspect those: `cargo doc -p hymeko_wasm --target wasm32-unknown-unknown`.
- Daemon / IPC crates (`hymeko_daemon`, `hymeko_client`, `hymeko_mcp`) are excluded for similar reasons (iceoryx2 + MCP transport dependencies).
